## The idea

Ever change a line in your system prompt and suddenly everything breaks?

**Showdown** makes it easy to verify that your AI model produces expected results.

1. Define an expected output
2. Provide your system prompt and example user input
3. Test against any model on OpenRouter and see which models pass, alongside latency and token use.
